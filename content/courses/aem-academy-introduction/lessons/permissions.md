Not all users have the same level of access within AEM.

Typical permission levels include:

## Author

- Create and edit pages
- Add and configure components
- Upload assets
- Submit content for review

## Publisher / Approver

- Review content
- Approve content
- Publish pages to live environments

## Administrator / Developer

- Configure templates
- Create or modify components
- Manage system-level settings

Permissions protect the platform from accidental changes and ensure accountability.
