All imagery and digital files are stored in AEM Assets.

Assets may include:

- Images
- PDFs
- Videos

When building a page, you do not upload media directly into a component. Instead:

1. Assets are uploaded to the appropriate folder structure
2. They are named and tagged correctly
3. They are selected and linked within components

Best practice when working with assets and to be agreed before a project kicks off:

- Follow agreed naming conventions
- Avoid duplicating files
- Optimise images before upload (to help with page speed)
- Use the correct file format

A clean asset structure makes content easier to manage long term.
