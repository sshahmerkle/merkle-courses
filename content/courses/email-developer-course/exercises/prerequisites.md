---
title: Prerequisites
order: 1.5
duration: 30 min
difficulty: easy

objectives:
  - Download and install a development environment - Visual Studio Code & Sublime Text are recommended.
  - Create Gmail, Outlook.com & Yahoo! Mail mailboxes
  - Create an account on Postdrop or Litmus Putsmail
  - Create an account on Figma
    
---

Prior to beginning this course, it is advised to fulfill the prerequisites. In order to ensure a comprehensive and effective testing process, it is necessary to have test inbox accounts. Postdrop & Litmus Putsmail are valuable tools for testing email templates, as it enables you to send HTML tests to your designated inboxes.
